# Claude Installer

Быстрая установка Claude Code с поддержкой любого Anthropic API-провайдера.

## Возможности

- 🚀 Установка Claude Code
- 🔄 Обновление Claude
- 🌍 Любой Base URL
- 🔑 Любой API Key
- 🌐 HTTP / SOCKS5 Proxy
- 🗑️ Полное удаление
- ⚙️ Изменение API без переустановки

## Установка

```bash
bash <(curl -fsSL raw.githubusercontent.com/D00MSH0P/claude-installer_ot_-tpbratvpn/refs/heads/main/claude_install.sh)
```

## Требования

- Ubuntu / Debian
- Root
- Интернет
